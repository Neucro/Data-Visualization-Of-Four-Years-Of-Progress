# Data-Visualization-Of-Four-Years-Of-Progress
In this data analysis, I made a dataset by my own from some old result sheets of my friend's bachelor's result and tried to grab some data insights from it by using matplotlib and pandas with python. It's a prgoress sheet of consquent four years of marks during a program called "Bachelor's of Science in Computer Science". In X axes, I kept all the courses and on Y axes I kept all the grade points. For normalization, I convert the course names into number regarding their level.
